AI Agent Portfolio

My portfolio is being built around two complementary systems: one knowledge/evidence agent and one action/workflow agent. The goal is to show real AI Agent engineering depth: retrieval, citations, evaluation, tool use, state, memory, safety, and demo-ready documentation.

Traceable Agentic RAG System

Evidence-grounded research and QA agent · Current portfolio project

This project is not a generic ChatPDF app. It is designed as a traceable RAG workflow that can break down complex questions, retrieve evidence from multiple angles, select and cite sources, and evaluate whether the generated answer is actually supported.

Latest build update · 2026-06-09: added configurable slow-retrieval threshold handling for dashboard latency cards. The Traceable RAG trace can now show total/average retrieval time and mark a run as warn when retrieval exceeds the selected threshold. Current status: in-progress observability and evaluation infrastructure; no public demo is claimed yet.

Deep Agent Workbench

LangChain Deep Agents-style autonomous workflow agent · Current portfolio project

This project focuses on action-oriented agents: systems that do not just answer once, but plan, use tools, inspect results, remember intermediate context, ask for approval when needed, and continue until the task is complete or safely stopped.

Core workflow

  • High-level task planning
  • Tool selection and execution
  • Working memory / state
  • Result inspection
  • Retry and recovery paths

Safety and reliability

  • Human approval for risky actions
  • Sandboxed execution target
  • Structured tool schemas
  • Execution traces
  • Task-success regression tests
